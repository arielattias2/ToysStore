Ariel Toys api documentation

Sign Up:/users (POST)
  • user (required)
  • email (required)
  • password (required)

    • Login:/users/login (POST)
  • email (required)
  • password (required)

    • Authenticate user info:/users/me (GET)

  • Get all toys (diffult page and limit, 0 and 10):/toys (GET)
  • Get single toy by id:/toys/single/:id (GET)
  • Get all toys by category:/toys?category=category name (GET)
  • Get 10 toys by page number:/toys?page=number (GET)
  • Get X toys in each page:/toys?limit=number (GET)
  • Search toys by name or info:/toys?q=search query (GET)

    • Get product - (only for register user - must send user's auth-token through headers):/toys (POST)
  • name (required)
  • info (required)
  • category (required)
  • img_url
  • price (required)

    • Edit product - (only for register user - must send user's auth-token through headers):/toys/:id (PUT)
  • name (required)
  • info (required)
  • category (required)
  • img_url
  • price (required)

    • Delete product - (only for register user - must send user's auth-token through headers):/toys/:id (DELETE)
